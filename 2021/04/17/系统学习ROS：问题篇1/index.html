我在第一次跑程序时遇到的问题：

（1）我的环境：

  • ROS melodic桌面完整版

  • 根据教程配置好的VS Code

  • Ubuntu内装有anaconda

  • Ubuntu自带python2、python3

（2）问题：

写好Client节点后，运行launch文件报错，并且只出现一只乌龟。

（3）解决过程
